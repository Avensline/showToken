# Show Token - an add-on for Owlbear Rodeo

An add-on for the GM that allows you to show selected players (or everyone) a zoomed-in
preview of the selected token in a separate window - without having to search for it on the board.

## How It Works

1. The GM selects a token on the board, right-clicks (or clicks the three dots), and selects **“Show Token”**.
2. A window opens in the center of the screen with a list of players in the room-you can select one,
   several, or check **“All Players”**.
3. After clicking **“Show Token”**, a window with a large preview of the token opens for the selected recipients (and for the GM themselves, as a
   confirmation preview).
4. Close the window by clicking the **✕** button in the upper-right corner.

The button in the context menu is visible only to the GM—the GM decides who
to show the token to.
